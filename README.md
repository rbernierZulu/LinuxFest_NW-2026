Scaling PostgreSQL; Citus vs TimescaleDB, Choosing The Best-Fit
---

Anybody with a few years experience as a PostgreSQL DBA has heard of the CitusDB and TimescaleDB extensions. And for the repository and GitHub savvy individual it's a straightforward process getting and installing these fantastic extensions into your database system. But here's the question, which one?

This session guides you to the better suited solution. We'll cover the issues you need to consider and the underlying reasons selecting one technology over the other.
